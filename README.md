### Grupo7 | Punta Tombo | 1º cuatrimestre 2026

Este sitio web fue construido en el espacio de la materia Computación III, cátedra Wischñevsky, UNLa, para la carrera de Diseño y Comunicación Visual. El objetivo fue rediseñar una página web con problemas. Nos tocó la temática turismo, y elegimos rehacer puntatombo.com, porque un lugar tan hermoso merecía una página más actual y atractiva.
Nos propusimos rediseñar la identidad visual completa, con un estilo más relajado y amigable con posibles familias de turistas. Intentamos lograr una página dinámica, con cositas con las que el visitante pueda interactuar, y algunas sorpresas.

**Integrantes:** Cecilia Gómez | Candela Benavídez | Maira Torrez | Mariana Zaro

**Qué hicimos:**
- Una página con 5 htmls: index, guía del turista, galería, tienda online y contacto
- Responsive para 768px en todo el sitio
- Responsive para laptops 1366px sólo en algunas partes que lo requerían
- Usamos html5 + css + javascript; galerías con fancybox
- Probamos algunas cosas más complejas, algunos scripts extra y animaciones con keyframes

**Notas responsive**
- El widget del clima es externo, por tanto algunos tamaños tipográficos no pueden modificarse ya que vienen por defecto
